Vue 프로젝트 생성
Vue Project(Application) 생성 (Vite 기반 빌드)
1. npm create vue@latest
프로젝트 폴더 이동
2. cd vue-project
패키지 설치
3. npm install
Vue 프로젝트 서버 실행
npm run dev (py manage.py runserver 랑 같음)
4. 실습 할때 처음 들어가서 위에 꺼 다 하고 서버 실행에서 src랑 component안에 들어있는 기존 vue 삭제 후 새로 만들어서 함 
5. App.vue, ParentsPage.vue, ChildrenPage.cue, main.js 등등 수정
