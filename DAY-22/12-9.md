## 게시글 목록 출력
1. 게시글을 출력하는 컴포넌트를 연결하는 ArticleView의 route관련 코드 주석 해제하기
2. App 컴포넌트에 ArticleView 컴포넌트로 이동하는 RouterLink 작성
3. 게시글 목록을 담당하는 ArticleList 컴포넌트를 ArticleView 컴포넌트에 등록하기
4. store에 임시 데이터 articles 배열 작성하기
5. ArticleList 컴포넌트에서 게시글 목록 출력
   store의 articles 데이터 참조
   v-for 를 활용하여 하위 컴포넌트에서 사용할 article 단일 객체 정보를 props로 전달
6. ArticleListItem 컴포넌트는 내려 받은 props 를 정의
메인 페이지에서 게시글 목록 출력 확인

## DRF와의 요청과 응답
## DRF로 부터 응답 데이터 받기
1. DRF 서버로의 AJAX 요청을 위한 axios 설치 및 관련 코드 작성
npm install ( Vue 서버 종료 -> 설치 -> 서버 재실행 )
2. DRF 서버로 요청을 보내고 응답 데이터를 처리하는 getArticles 함수 작성
3. ArticleView 컴포넌트가 마운트 될 때 getArticles 함수가 실행되도록 함
-> 해당 컴포넌트가 렌더링 될 때 항상 최신 게시글 목록을 불러오기 위함
4. Vue와 DRF 서버를 모두 실행한 후 응답 데이터 확인
5. 그런데 DRF 서버 측에서는 문제 없이 응답했음
-> 서버는 응답했으나 브라우저 측에서 거절한 것
6. 브라우저가 거절한 이유
-> 'localhost:5173' 에서 '127.0.0.1:8000/api/v1/articles/'의 XFMLHttpRequest에 대한 접근이
