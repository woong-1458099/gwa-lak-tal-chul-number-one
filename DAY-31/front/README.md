# T.C. Front (Vue)

관통 프로젝트에서 프론트엔드 폴더(front/)로 바로 넣어 쓸 수 있게 재배치한 버전입니다.

## 실행

```bash
npm install
npm run dev
```

## 라우팅
- /home : 홈(히어로)
- /login : 로그인
- /signup : 회원가입
- /map : 지도
- /recommend : 추천결과 (추천 입력 페이지가 따로 없어서 임시로 연결)

## 환경변수
- `.env.example` 참고해서 `.env`를 만들고 `VITE_API_BASE_URL`을 백엔드 주소로 맞추세요.
