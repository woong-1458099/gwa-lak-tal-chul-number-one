# 우주괴물 문제 풀이 최연웅이 최연웅 2 한테 설명하듯이

# 1) 문제 요약
- N×N 격자
  - 0: 빈 칸, 1: 벽, 2: 우주괴물
- 각 괴물은 상/하/좌/우로 광선을 발사한다.
- 광선은 벽(1)을 만나면 그 방향 탐색을 즉시 중단한다.
- 괴물의 광선이 닿지 않은 빈칸의 개수를 구한다.

# 2) 접근 방식
- 델타 배열(상/하/좌/우)로 한방향으로 한칸씩 전진
  1. 괴물 좌표를 전부 수집한 후 광선을 처리한다.
     - 이유: 진행 중 2(괴물)를 3으로 덮어쓰면, 그 괴물의 차후 광선 발사가 무효가 되는 거 방지해야됨
  2. 각 방향 탐색은 괴물 칸 다음 칸(k=1) 부터 시작한다.
     - 괴물 자신의 칸을 3으로 바꾸지 않기 위함
  3. 빈 칸(0)만 3으로 마킹한다.
     - 벽(1) -> 탐색 즉시 종료
     - 괴물(2) / 이미 맞은 칸(3) -> 값 변경 없이 통과 가능(문제에서 벽에만 막힌다고 적혀있음)

# 3) 알고리즘/개념
- 2차원 배열 인덱싱, 델타 배열

# 4) 문제 풀이 순서
1. 입력으로 격자 구성
2. 모든 괴물 좌표를 목록에 저장
3. 목록의 각 괴물에 대해 4방향으로:
   - 다음 칸부터 한 칸씩 진행
   - 격자 밖이면 종료
   - 벽(1) 만나면 종료
   - 빈 칸(0) → 3으로 마킹
   - 괴물(2)나 3은 통과만 하고 계속 진행
4. 전체 격자에서 값이 0인 칸의 개수를 세어 출력


# 5) 주의 사항
- k=0부터 시작하면 괴물 자리를 3으로 마킹하는 실수 -> k=1부터
- 광선 처리 중 2를 3으로 덮어쓰기 -> 이후 그 괴물이 사라져 탐색 누락 ->반드시 괴물 좌표를 먼저 수집
- 벽을 지나 계속 진행하는 실수 -> 1을 만나면 즉시 break
