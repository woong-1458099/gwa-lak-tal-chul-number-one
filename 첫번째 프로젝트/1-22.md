# 이미 만들어진 작업 항목에 설명 템플릿 적용하기(새로 만드는 작업은 자동 적용)

해당 작업의 “키”를 클릭하면 아래와 같은 화면이 나옵니다.



설명란을 클릭하기 전에 번개모양 (회원정보 확인 제목 밑에 버튼)을 클릭합니다.



이미 만들어진 업무 항목 필드 편집을 누르고 설명 편집을 클릭합니다.



작업 1, 작업 2는 체크박스로 만들 수 있는데 방법은 설명 영역을 클릭 후 텍스트 편집기로 들어간 뒤에

[    ]  뒤에 띄어쓰기 없이 작업내용을 딱 붙여서 입력하시고 그 후에 [  ] 와 글자 사이에서 스페이스바를 누르시면 체크박스로 변합니다.
